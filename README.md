## 배포 방법

1. package.json version 설정
2. padkage.json description / repository url / keywords 수정
3. src 폴더내에 컴포넌트 셋팅
4. src > package > index.tsx 에 export로 반환 셋팅
5. npm publish 명령어로 배포
